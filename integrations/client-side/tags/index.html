CLIENT SIDE

Tags

JavaScript tag integrations send data to vendors directly from a client’s website.

Summary

Tags are code snippets, provided by technology vendors (analytics, marketing, personalization, contact solutions). After installing the Tealium Universal Tag utag.js library on a webpage, loading additional JavaScript does not require making a request to developers.

The Tealium iQ Tag Marketplace provides hundreds of vendors tags that may be added to your profile. With a data layer defined, variables are passed to any configured tag using Tealium’s data mapping interface

The integrations in the Tag Marketplace contains the code needed to communicate with the vendor and using Tealium’s universal data mappings.

Benefits

Benefits of Tealium tag integrations include:

  • A “plug and play” experience for all stakeholders, including developers and engineers.
  • Vendor code is deployed to exact specifications by mutual customers using Tealium.

Tag Features

Enhanced Tags

Enhanced tags send events directly to Tealium Customer Data Hub (CDH). For example, a user experience vendor inferring user sentiment by tracking mouse movements. These events are in addition to the actual tracking (view and link) calls made by utag.js. Clients maintain full control over their event volume by turning events on/off via the tag configurations.

The following example function sends data to the Tealium event endpoint:

window.tealium.sendEvent = function(event_data) {
    //Send Event to CDH event endpoint
    event_data['tealium_event'] = "EVENT_NAME";
    event_data['tealium_account'] = b.tealium_account; // required
    event_data['tealium_profile'] = b.tealium_profile; // required
    event_data['tealium_visitor_id'] = b.tealium_visitor_id; // required
    event_data['tealium_trace_id'] = b["cp.trace_id"]; // optional
    event_data['tealium_datasource'] = 'DATASOURCE_KEY'; // optional

    var data = new FormData();
    data.append("data", JSON.stringify(event_data));

    if (window.utag && utag.data && utag.data.tealium_account) {
        var xhr = new XMLHttpRequest();
        xhr.open("POST", "https://collect.tealiumiq.com/event?");
        xhr.send(data);
        }
    return true;
};

Synchronous vs. Asynchronous

The Universal Tag (utag.js) provides clients with support for synchronous and/or asynchronous vendor tags formats.

Cookie Match Tags

Cookie match tags are a special type of JavaScript tag integration that relate to server-side connectors. Tealium initiates a request to the vendor, typically at the start of each session. This request includes Tealium’s visitor ID and additional parameters used in the redirect URL.

Parameter Description
tealium_visitor_id (Required) An anonymized and unique identifier for the visitor associated with the event
tealium_account The name of your Tealium account
tealium_profile The name of your Tealium profile
tealium_trace_id (Optional) For use with Trace
tealium_datasource (Optional) The data source key

Learn more in the Google Developer’s Cookie Matching Guide.

Example: Cookie Match Request

The following example demonstrates a cookie match request:

GET https://sync.example-dsp.net/get-userId?tealium_visitor_id=abxyz0909&tealium_account=egbrand&tealium_profile=main&tealium_trace_id=09454&tealium_datasource=0x7jg

This client-initiated request pings the DSP to return a user ID and forwards the DSP vendor’s user ID and Tealium’s visitor ID back to Tealium. This indicates that Tealium is hosting the match table, which is preferred. The final request to Tealium includes all of the provided Tealium variables and the DSP matched ID.

Example: Redirect to Tealium

The following example demonstrates a redirect to Tealium:

GET https://collect.tealiumiq.com/event?dsp_uid=4595-23423442&tealium_visitor_id=abxyz0909&tealium_account=egbrand&tealium_profile=main&tealium_trace_id=09454&tealium_datasource=0x7jg1

Example: Cookie Match Request with Redirect to Tealium

The following cookie match example passes the redirect request as a URL parameter, if the DSP’s endpoint accepts it:

GET https://sync.example-dsp.net/get-userId?redirect=https://collect.tealiumiq.com/event?dsp_uid=$UID&tealium_visitor_id=abxyz0909&tealium_account=egbrand&tealium_profile=main&tealium_trace_id=09454&tealium_datasource=0x7jg
