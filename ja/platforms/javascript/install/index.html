インストール

タグ管理とユーザーアクティビティのトラッキングのために、Tealiumをウェブサイトに追加します。

ユニバーサルデータオブジェクト（utag_data）

ユニバーサルデータオブジェクト（UDO）は、utag_dataと呼ばれるJavaScriptオブジェクトです。このオブジェクトでは、ウェブページからの動的データがTealiumタグに渡されます。 このオブジェクトのプロパティには、ビジネスに合わせてベンダーに依存しないわかりやすい名前を使用します。  

ユニバーサルタグを読み込む前に、utag_dataを定義して値を入力するようにしてください。

例：

<!-- Tealium Universal Data Object -->
<script type="text/javascript">
var utag_data={
    "page_type"     : "section",
    "site_section"  : "men",
    "page_name"     : "Men's Fashion | Acme Inc.",
    "country_code"  : "US",
    "currency_code" : "USD"
};
</script>

ユニバーサルタグ（utag.js）

JavaScriptライブラリであるutag.jsは、Tealiumのユニバーサルタグで、複数のサードパーティタグを1つのコードスニペットに置き換えることができます。 utag.jsファイルには、サードパーティタグをサイトに読み込むのに必要なコードがすべて生成済みの状態で含まれています。

<!-- Tealium Universal Tag -->
<script type="text/javascript">
(function(a,b,c,d){
a='//tags.tiqcdn.com/utag/ACCOUNT/PROFILE/ENVIRONMENT/utag.js';
b=document;c='script';d=b.createElement(c);d.src=a;
d.type='text/java'+c;d.async=true;
a=b.getElementsByTagName(c)[0];a.parentNode.insertBefore(d,a)
})();
</script>

ファイルへのパスは、以下の属性で構成されています。

パラメータ 詳細
account アカウント名。
profile プロファイル名。
environment “dev”、”qa”、”prod”のいずれか。

アカウント用のタグ スニペット はコードセンターから取得できます。

その他の参考情報：

  • utag.jsの機能
  • utag.sync.jsの使用

コードの配置

Tealiumコードは、（<head>タグではなく）<body>の開始タグの後に配置される設計になっています。 この場所に配置することで、最大多数のベンダーとの互換性を実現し、訪問者が次のページに移動する前に、サードパーティのトラッキングを完了させることが可能になります。utag.jsコードは<body>タグ内のどこにでも配置できますが、一番上に配置することを推奨しています。 ページでのutag.jsの読み込み開始が早ければ、ページのレンダリング完了後のタグ実行準備も早くできます。
