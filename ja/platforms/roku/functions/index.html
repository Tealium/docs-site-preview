ROKU

関数の定義

Tealium Rokuライブラリの関数の定義です。

このセクションでは、Tealium for Rokuライブラリで提供されている関数の構文を示します。 Roku用アプリは、強力なスクリプト言語であるBrightScriptで作成されています。 詳細は、RokuのBrightScript言語リファレンスを参照してください。

TealiumBuilder()

Tealiumオブジェクトの初期化に役立つビルダー関数。 ユーティリティ関数を使って任意のパラメータを設定します。 Build()を呼び出してTealiumオブジェクトを作成します。

TealiumBuilder(account as String, profile as String, logLevel as Integer) as Object
パラメータ 型 詳細
account 文字列 Tealiumアカウントの名前
profile 文字列 Tealiumプロファイルの名前（デフォルト：”main”）
environment (optional) 文字列 Tealium環境：”qa”または”prod”
datasource (optional) 文字列 UDHのデータソースキー。例： “abc123”

例：

builder = TealiumBuilder("ACCOUNT", "PROFILE", 1)
builder.SetEnvironment("prod")
builder.SetDatasource("DATASOURCE")
teal = builder.Build()

createTealium()

イベントのトラッキングに使用されるTealiumオブジェクトを作成します。

createTealium(account as String, profile as String, env as String, logLevel as Integer) as Object
パラメータ 型 詳細
account 文字列 Tealiumアカウントの名前
profile 文字列 Tealiumプロファイルの名前（通常は”main”）
env 文字列 Tealium環境：”qa”または”prod”
log level 整数 ログレベル：0 = なし、1 = エラー、2 = 警告、3 = すべて

例：

teal = createTealium("account", "profile", "env", 3) 

trackEvent()

関連付けられたデータを含むイベントをトラッキングし、任意でコールバック関数をトリガーします。

trackEvent(eventType as String, eventName as String, data as Object, callback as Object) as Void
パラメータ 型 詳細
eventType 文字列 Tealiumのイベントの種類（アクティビティ、コンバージョン、インタラクション、ビュー）
eventName 文字列 イベントの名前（UDHではevent_name属性になります）
data オブジェクト コンテクスチュアルイベントデータである、キーと値を含む”roAssociativeArray”オブジェクト
callback オブジェクト “roEvent”引数を受け取る”callback”という名前の関数プロパティを持つオブジェクト。 このコールバック関数は、TrackEventコールの完了時にトリガーされます。

例：

data = CreateObject("roAssociativeArray")
data.testKey = "testValue"

cb = CreateObject("roAssociativeArray")
cb.callBack = Function(event)
    print "Callback called!"
End Function
teal.trackEvent("activity", "someEvent", data, cb) 

resetSessionId()

リセットして新しいセッションIDを返します。 このセッションIDは、ウェブサイトセッションと同様に、現在のユーザーセッションを識別しますが、有効期限がありません。 createTealium()が呼び出されるたびに、新しいセッションIDが作成されます。 

resetSessionId() as Integer

toStr()

現在のTealiumインスタンスの構成の詳細を含む判読可能な文字列を返します。例： “Tealium instance for account: ACCOUNT profile: PROFILE environment: ENV”

toStr() as String

例：

print teal.toStr()
