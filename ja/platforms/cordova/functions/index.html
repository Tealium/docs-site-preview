CORDOVA

関数の定義

Tealium for Cordovaプラグインの関数の定義です。

構成

init()

Tealium Cordovaプラグインを初期化します。

パラメータ データ型 詳細/使用方法
config JavaScriptオブジェクト* 別の表を参照
successCallback 関数 デフォルトの成功コールバックをオーバーライドして成功した場合の処理のカスタマイズを許可します
errorCallback 関数 デフォルトの成功コールバックをオーバーライドして成功した場合の処理のカスタマイズを許可します

* configオブジェクト

パラメータ データ型 詳細/使用方法
account 文字列 Tealiumアカウントの名前
例：”tealium”
profile 文字列 Tealiumプロファイルの名前
例：”main”
environment 文字列 Tealium環境の名前
例：”dev”
instance 文字列 任意のTealiumインスタンス名
例：”main”
dataSourceId 文字列 任意の英数字からなるデータソースキー
例：”3obb8c”
isLifecycleEnabled 文字列 自動的なライフサイクルトラッキングを有効または無効にします
例：”false”
logLevel 整数（列挙型） Cordovaプラグインのログレベルを設定します（コアSDKのログレベルには影響しません）。 指定されていない場合は、デフォルトでtealium.logLevels.DEVになります。
collectDispatchURL 文字列 Tealium CollectのデフォルトのディスパッチURLをオーバーライドします
例：https://collect.tealiumiq.com/vdata/i.gif?tealium_account=tealium&tealium_profile=mobile
collectDispatchProfile 文字列 Tealium Collectのデフォルトのディスパッチプロファイルをオーバーライドします（デフォルトでは常に”main”）
例：”mobile”
isCrashReporterEnabled 文字列 設定した場合、Crash Reporterモジュールが有効になります（デフォルトでは無効）
例：”true”

トラッキング

trackView()

Tealium Cordovaプラグインにスクリーンビューのトラッキングリクエストを送信します。

パラメータ データ型 詳細/使用方法
Data Object JavaScriptオブジェクト このトラッキングコールのデータレイヤーに、指定されたキーと値のペアが設定されます。

例：
{"tealium_event" : "page_view"}
instance 文字列 任意のインスタンス名の定数。特定のトラッキングインスタンスを参照する際に使用されます。

例：
“main”

trackEvent()

Tealium Cordovaプラグインにイベントのトラッキングリクエストを送信します。

パラメータ データ型 詳細/使用方法
Data Object JavaScriptオブジェクト このトラッキングコールのデータレイヤーに、指定されたキーと値のペアが設定されます。

例：
{"tealium_event": "Click to call"}
instance 文字列 任意のインスタンス名の定数。特定のトラッキングインスタンスを参照する際に使用されます。

例：
“main”

track()

Tealium Cordovaプラグインにトラッキングリクエストを送信します。

パラメータ データ型 詳細/使用方法
Event Type 文字列 Tealiumイベントの種類

例：
“link”（イベント）または”view”（スクリーンビュー）
Data Object JavaScript/JSONオブジェクト このトラッキングコールのデータレイヤーに、指定されたキーと値のペアが設定されます。

例：
{"tealium_event" : "page_view"}
instance 文字列 任意のインスタンス名の定数。特定のトラッキングインスタンスを参照する際に使用されます。

例：
“main”

データ管理

addPersistent()

Tealiumの永続性データストアにデータを追加します。

パラメータ データ型 詳細/使用方法
Key Name 文字列 永続化する値のキー名

例：
“user_hashed_email”
値 文字列または文字列のarray このキーのために永続化する値

例：
“293c2529495b45cbbc7b451df5720e8c”
instance 文字列 任意のインスタンス名の定数。特定のトラッキングインスタンスを参照する際に使用されます。

例：
“main”

addVolatile()

Tealiumの揮発性データストアにデータを追加します。

パラメータ データ型 詳細/使用方法
Key Name 文字列 揮発性メモリに格納する値のキー名
例：”user_hashed_email”
値 文字列または文字列のarray、オブジェクトまたはディクショナリ このキーのために揮発性メモリに格納する値
例：”293c2529495b45cbbc7b451df5720e8c”
instance 文字列 任意のインスタンス名の定数。特定のトラッキングインスタンスを参照する際に使用されます。
例：”main”

Tealiumの永続性データストアからデータを削除します。

パラメータ データ型 詳細/使用方法
Key Name 文字列 永続性ストレージから削除する値のキー名

例：
“user_hashed_email”
instance 文字列 任意のインスタンス名の定数。特定のトラッキングインスタンスを参照する際に使用されます。

例：
“main”

removeVolatile()

Tealiumの揮発性データストアからデータを削除します。

パラメータ データ型 詳細/使用方法
Key Name 文字列 揮発性メモリから削除する値のキー名

例：
“user_hashed_email”
instance 文字列 任意のインスタンス名の定数。特定のトラッキングインスタンスを参照する際に使用されます。

例：
“main”

getPersistent()

Tealiumの永続性データストアから値を取得します。戻り値は、最初に格納されたのと同じデータ型（文字列または文字列のarray）の永続性データです。

パラメータ データ型 詳細/使用方法
Key Name 文字列 永続性ストレージから取得する値のキー名

例：
“user_hashed_email”
instance 文字列 任意のインスタンス名の定数。特定のトラッキングインスタンスを参照する際に使用されます。

例：
“main”

getVolatile()

Tealiumの揮発性データストアから値を取得します。戻り値は、最初に格納されたのと同じデータ型（文字列、文字列のarray、オブジェクト/ディクショナリ）の格納されているデータです。

パラメータ データ型 詳細/使用方法
Key Name 文字列 揮発性ストレージから取得する値のキー名

例：
“user_hashed_email”
instance 文字列 任意のインスタンス名の定数。特定のトラッキングインスタンスを参照する際に使用されます。

例：
“main”

addRemoteCommand()

Tealium iQによって開始される、後で実行するためのリモートコマンドを追加します。

パラメータ データ型 詳細/使用方法
リモートコマンド ID 文字列 リモートコマンドのID。 コードブロックをトリガーするには、一致するIDをTealium iQで使用する必要があります。

例：
“changeBackgroundColor”
instance 文字列 任意のインスタンス名の定数。特定のトラッキングインスタンスを参照する際に使用されます。

例：
“main”
Code block 関数 関数リテラルまたは関数参照。これは、Tealium iQから後でリモート実行する場合に利用できます。

例：
function (message) { //do something}

getVistorId()

現在のユーザーの現在のTealium訪問者IDを返します。 パラメータは使用しません。 戻り値は、現在の訪問者IDの文字列表現です。
