REACT NATIVE

関数の定義

Tealium for React Nativeの関数定義のリストです。

initialize()

他のメソッドを呼び出す前にTealiumを初期化します。

initialize(account,
    profile,
    env,
    ios_datasource_key,
    android_datasource_key);
パラメータ 型 詳細
account 文字列 Tealiumアカウントの名前
profile 文字列 Tealiumプロファイルの名前（デフォルト：”main”）
environment 文字列 Tealium環境：”qa”または”prod”
iOS data source (optional) 文字列 UDHのデータソースキー。例： “abc123”
Android data source (optional) 文字列 UDHのデータソースキー。例： “def456”

initializeWithConsentManager()

他のメソッドを呼び出す前に同意管理でTealiumを初期化します。

initializeWithConsentManager(account,
        profile,
        env,
        ios_datasource_key,
        android_datasource_key);
パラメータ 型 詳細
account 文字列 Tealiumアカウントの名前
profile 文字列 Tealiumプロファイルの名前（デフォルト：”main”）
environment 文字列 Tealium環境：”qa”または”prod”
iOS data source (optional) 文字列 UDHのデータソースキー。例： “abc123”
Android data source (optional) 文字列 UDHのデータソースキー。例： “def456”

initializeCustom()

他のメソッドを呼び出す前に同意管理でTealiumを初期化します。

initializeCustom(account,
        profile,
        env,
        ios_datasource_key,
        android_datasource_key,
        instance_name,
        true,
        null,
        null,
        true,
        true);
パラメータ 型 詳細
account 文字列 Tealiumアカウントの名前
profile 文字列 Tealiumプロファイルの名前（デフォルト：”main”）
environment 文字列 Tealium環境：”qa”または”prod”
iOS data source (optional) 文字列 UDHのデータソースキー。例： “abc123”
Android data source (optional) 文字列 UDHのデータソースキー。例： “def456”
instance name 文字列 Tealiumインスタンスの名前（例：任意の文字列を指定可能）
enable life cycle ブーリアン ライフサイクルトラッキングを有効にする（デフォルト：true）
override publish settings URL 文字列 オーバーライドする場合は公開設定のURLを表す文字列、それ以外の場合はnull（デフォルト：null）
override tag management URL 文字列 オーバーライドする場合はタグ管理URLを表す文字列、それ以外の場合はnull（デフォルト：null）
enable collect endpoint ブーリアン trueの場合は、データがCollectエンドポイントに送信されます（デフォルト：true）
enable consent manager ブーリアン trueの場合は、同意管理が有効になります

trackEvent()

イベント名とイベントデータでイベントをトラッキングします。

trackEvent(eventName, data);
パラメータ 型 詳細
eventName 文字列 イベントの名前（UDHではevent_name属性になります）
data オブジェクト キーと値のペアのJSONオブジェクト

trackEventForInstanceName()

アプリにTealiumのインスタンスが複数ある場合にイベントをトラッキングします。

trackEvent(instanceName, eventName, data);
パラメータ 型 詳細
instanceName 文字列 Tealiumインスタンスの名前
eventName 文字列 イベントの名前（UDHではevent_name属性になります）
data オブジェクト キーと値のペアのJSONオブジェクト

trackView()

スクリーン名とビューデータでビューをトラッキングします。

trackView(screenName, data);
パラメータ 型 詳細
screenName 文字列 イベントの名前（UDHではscreen_title属性になります）
data オブジェクト キーと値のペアのJSONオブジェクト

trackViewForInstanceName()

アプリにTealiumのインスタンスが複数ある場合にビューをトラッキングします。

Tealium.trackViewForInstanceName(instanceName, screenName, data);
パラメータ 型 詳細
instanceName 文字列 Tealiumインスタンスの名前
screenName 文字列 イベントの名前（UDHではscreen_title属性になります）
data オブジェクト キーと値のペアのJSONオブジェクト

setVolatileData()

アプリが終了するまで後続の各イベントやビューと一緒に送信される揮発性データを設定します。この場合、データはキーと値のペアから成るJSONオブジェクトで、キーは文字列、値は文字列または文字列のarrayになります。

Tealium.setVolatileData(data);

setVolatileDataForInstanceName()

アプリが終了するまで後続の各トラッキングコールに含まれる揮発性データを設定します。 アプリにTealiumのインスタンスが複数ある場合に使用します。

Tealium.setVolatileData(instanceName, data);

getVolatileData()

渡されたキーの値を取得します。

Tealium.getVolatileData(key, value);

getVolatileDataForInstanceName()

渡されたインスタンスとキー名の揮発性データの値を取得します。アプリにTealiumインスタンスが複数ある場合に使用します。

Tealium.getVolatileData(instanceName, key, value);

setPersistentData()

アプリを終了し、再度起動しても後続の各イベントやビューと一緒に送信される永続性データを設定します。この場合、データはキーと値のペアから成るJSONオブジェクトで、キーは文字列、値は文字列または文字列のarrayになります。

Tealium.setPersistentData(data);

setPersistentDataForInstanceName()

アプリを終了し、再度起動しても後続の各イベントやビューと一緒に送信される永続性データを設定します。この場合、データはキーと値のペアから成るJSONオブジェクトで、キーは文字列、値は文字列または文字列のarrayになります。 アプリにTealiumのインスタンスが複数ある場合に使用します。

Tealium.setPersistentData(instanceName, data);

removeVolatileData()

Tealium.setVolatileData()を使用して以前設定された揮発性データを、キー名のarrayを受け取って削除します。

Tealium.removeVolatileData(keys);

removeVolatileDataForInstanceName()

キー名のarrayを受け取ってsetVolatileData()を実行します。 アプリにTealiumのインスタンスが複数ある場合に使用します。

Tealium.removeVolatileData(instanceName, keys);

removePersistentData()

Tealium.setPersistentData()を使用して以前設定された永続性データを、キー名のarrayを受け取って削除します。

Tealium.removePersistentData(keys);

removePersistentDataForInstanceName()

Tealium.setPersistentData()を使用して以前設定された永続性データを、キー名のarrayを受け取って削除します。 アプリにTealiumのインスタンスが複数ある場合に使用します。

Tealium.removePersistentData(instanceName, keys);

getVisitorID()

ユーザーの訪問者IDを取得します。 コールバックを渡してこの訪問者IDを使用します。

Tealium.getVisitorID(visitorID);

getVisitorIDForInstanceName()

ユーザーの訪問者IDを取得します。 コールバックを渡してこの訪問者IDを使用します。 アプリにTealiumのインスタンスが複数ある場合に使用します。

Tealium.getVisitorID(instanceName, visitorID);

getUserConsentStatus()

ユーザーの同意ステータスを取得します。 コールバックを渡してuserConsentStatusを使用します。

Tealium.getUserConsentStatus(userConsentStatus);

getUserConsentStatusForInstanceName()

ユーザーの同意ステータスを取得します。 コールバックを渡してuserConsentStatusを使用します。 アプリにTealiumのインスタンスが複数ある場合に使用します。

Tealium.getUserConsentStatusForInstanceName(instanceName, userConsentStatus);

setUserConsentStatus()

ユーザーの同意ステータスを設定します。

Tealium.setUserConsentStatus(userConsentStatus);
値 詳細
0 不明
1 同意あり
2 同意なし
3 無効（Objective-Cのみ）

setUserConsentStatusForInstanceName()

ユーザーの同意ステータスを設定します。 アプリにTealiumのインスタンスが複数ある場合に使用します。

Tealium.setUserConsentStatusForInstanceName(instanceName, userConsentStatus);
値 詳細
0 不明
1 同意あり
2 同意なし
3 無効（Objective-Cのみ）

getUserConsentCategories()

ユーザーの同意カテゴリを取得します。 コールバックを渡してuserConsentCategoriesを使用します。

Tealium.getUserConsentCategories(userConsentCategories);

getUserConsentCatgoriesForInstanceName()

ユーザーの同意カテゴリを取得します。 コールバックを渡してuserConsentCategoriesを使用します。 アプリにTealiumのインスタンスが複数ある場合に使用します。

Tealium.getUserConsentCategoriesForInstanceName(instanceName, userConsentCategories);

setUserConsentCategories()

ユーザーの同意カテゴリを設定します。 カテゴリを設定するには、文字列のarrayを渡します。

Tealium.setUserConsentCategories(userConsentCategories);

setUserConsentCatgoriesForInstanceName()

ユーザーの同意カテゴリを設定します。 カテゴリを設定するには、文字列のarrayを渡します。 アプリにTealiumのインスタンスが複数ある場合に使用します。

Tealium.setUserConsentCategoriesForInstanceName(instanceName, userConsentCategories);

resetUserConsentPreferences()

ユーザーの同意ステータスと同意カテゴリをリセットします。

Tealium.resetUserConsentPreferences();

resetUserConsentPreferencesForInstanceName()

ユーザーの同意ステータスと同意カテゴリをリセットします。 アプリにTealiumのインスタンスが複数ある場合に使用します。

Tealium.resetUserConsentPreferencesForInstanceName(instanceName);

setConsentLoggingEnabled()

ユーザーの同意の記録を設定します。

Tealium.setConsentLoggingEnabled(enabled);

setConsentLoggingEnabledForInstanceName()

ユーザーの同意の記録を設定します。 アプリにTealiumのインスタンスが複数ある場合に使用します。

Tealium.setConsentLoggingEnabled(instanceName, enabled);

isConsentLoggingEnabled()

ユーザーの同意の記録が有効になっているかどうかを確認します。 コールバックを渡して同意の記録の値を使用します。

isConsentLoggingEnabled(enabled);

isConsentLoggingEnabledForInstanceName()

ユーザーの同意の記録が有効になっているかどうかを確認します。 コールバックを渡して同意の記録の値を使用します。 アプリにTealiumのインスタンスが複数ある場合に使用します。

Tealium.isConsentLoggingEnabled(instanceName, enabled);
